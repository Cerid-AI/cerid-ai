Manage Cerid AI knowledge base sync. Accept an argument:

- `status` (default): Run `python3 scripts/cerid-sync.py status` to compare local vs sync
- `export`: Run `python3 scripts/cerid-sync.py export` to export local KB to Dropbox sync dir
- `import`: Run `python3 scripts/cerid-sync.py import` for non-destructive merge from sync dir

Report artifact counts, timestamps, and any conflicts found.
